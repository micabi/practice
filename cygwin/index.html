windowsでの導入がとっても難しかった。

まずはCygwinをインストール

公式サイトから32bit版ならsetup-x86.exe、64bit版ならsetup-x86_64.exeをダウンロードする。

apt-cygを導入してコマンドラインからcygwinのパッケージをインストール

apt-cygという、Cygwinのパッケージをコマンドラインからインストールするためのパッケージを入手する。
https://github.com/kou1okada/apt-cygから zipファイルでダウンロードしてきて解凍し、C:\cygwin\usr\local\bin内にapt-cygというフォルダを作って、そこにおく。

apt-cygというフォルダに実行権限0755を与える。
Cygwinを起動して、cd /cygdrive/c/cygwin/usr/local/bin/apt-cyg
chmod 0755 apt-cyg

apt-cyg -m ftp://ftp.iij.ad.jp/pub/cygwin updateとしてレポジトリを国内に変更。

apt-cygを利用してtigをインストール

apt-cyg install tigを実行。

CygwinをXAMPPで使用する前提で、起動時にhtdocsを開くようにする。

環境変数のユーザー環境変数に、新規で変数名：HOME、変数値：/cygdrive/c/xampp/htdocsと設定。
もしくは、C:\cygwin\home\WATARU内にある.bashrcの末行にcd /cygdrive/c/xampp/htdocs

giboを導入

giboをzipファイルでダウンロードしてくる。
C:\Users\WATARUにbinフォルダを作ってgiboの中身を入れる。

ユーザー環境変数にパスC:\Users\WATARU\bin\giboを追加する。

cmdプロンプトでC:\Users\WATARU\binに移動してgibo -u

CygwinやMinnityでも同様にするが、時間がかかる場合がある。

hubを導入

hubの下の方にある「Standalone」のcompiled binariesから最新版をダウンロードし、
解凍して、C:\Users\WATARUにbinフォルダを作ってhubの中身を入れる。

そのhubフォルダの中にあるbinフォルダに対して、ユーザー環境変数にパスC:\Users\WATARU\bin\hub\binを追加する。

cmdプロンプトでC:\Users\WATARU\binに移動して、gibo --version。
